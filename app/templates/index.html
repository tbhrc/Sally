Sally Candidate Screening

Upload the requisition package to evaluate candidates. Provide a CSV or Excel workbook with the required columns and attach the resume files.

Input Expectations

  • Candidate sheet must include columns: candidate_id, full_name, email, years_experience, resume_filename.
  • Resume filenames must match the values in resume_filename (including file extension).
  • Supported resume formats: PDF, DOCX, or plain text.
